Introduction to Templates

What are Templates?

Templates are reusable structures that define the capabilities and hierarchy of workflows. They act as "vessels" that meet different requirements for creating workflows. Think of templates as blueprints that ensure consistency across multiple workflows.

Key Concept: Templates create the structure, workflows populate that structure with content, and boards execute the tasks.

Template Features

  • Up to 10 hierarchical levels - Define complex nested structures
  • 13 configurable property types - Customize what data each level can store
  • Reusable across workflows - One template, many workflows
  • Template validation - Ensure data integrity and consistency

Creating Your First Template

Step-by-Step Guide

1

Open Template Builder

Navigate to the Template Builder from the main menu or workflow page.

2

Click "Create New Template"

Start with a blank template or choose from existing templates to copy.

3

Name Your Template

Give your template a descriptive name (e.g., "GDPR Compliance Template").

4

Add Levels

Define your hierarchy by adding levels. Common patterns include:

  • Control Families → Controls → Requirements → Evidence
  • Functions → Categories → Subcategories → Actions
  • Phases → Tasks → Subtasks → Deliverables
5

Configure Properties

Enable the properties you need for each level (icons, descriptions, tags, etc.).

6

Save and Test

Save your template and test it by creating a sample workflow.

Designing Level Hierarchies

Understanding Levels

Levels represent the hierarchical structure of your workflow. Each level can have child levels, creating a tree-like structure up to 10 levels deep.

Level 1: Control Families
Level 2: Controls
Level 3: Requirements
Level 4: Evidence

Best Practices for Level Design

Do: Use clear, descriptive level names that match your domain
Don't: Use generic names like "Level 1", "Level 2"
Do: Keep levels logically related (parent-child relationships make sense)
Don't: Create too many levels (3-5 is usually optimal)
