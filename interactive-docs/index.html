Platform Overview

The Newflows platform uses a three-tier architecture designed for compliance management consultancy. Understand how Templates, Workflows, and Boards work together.

Templates

Create reusable structures that define workflow capabilities. Templates are the vessels that meet different requirements for creating workflows.

  • Up to 10 hierarchical levels
  • 13 configurable property types
  • Reusable across multiple workflows
  • Template validation and testing
Learn More

Workflows

Macro-level strategic maps for project management, strategies, manufacturing, and other pipelines. Built on top of templates.

  • Four creation methods
  • Creation and Execution modes
  • Workflow linking and synchronization
  • Export to boards
Learn More

Boards

Micro-level execution spaces where workflows land for task management. Kanban-style project management with full control.

  • Kanban-style columns
  • Dynamic lists and references
  • Milestones and categories
  • Bulk operations and groups
Learn More

Compliance Management Use Case

See how the platform works in real-world compliance management consultancy scenarios.

1

Consultants Create Templates

Design regulation-specific structures (GDPR, ISO 27001, NIST CSF, etc.) that define compliance requirement levels and data collection points.

2

Consultants Create Workflows

Apply templates to specific regulations, build comprehensive compliance maps, and share as reference guidelines with clients.

3

Export to Boards for Execution

Transfer workflows to execution environment. Consultants manage references while clients execute compliance tasks and track progress.
