Quick Start Guide

Get up and running with the Newflows platform in just 5 minutes. Follow this step-by-step guide to create your first template, workflow, and board.

5-Minute Quick Start

This guide will walk you through creating a simple compliance checklist workflow from start to finish.

1

Create Your First Template

Time: 2 minutes

  1. Navigate to the Template Builder
  2. Click "Create New Template"
  3. Name it "Simple Checklist"
  4. Add one level called "Item"
  5. Enable properties: Name, Description, Done checkbox
  6. Save the template
Tip: For your first template, keep it simple. You can always add more levels and properties later.
2

Create a Workflow from Template

Time: 1 minute

  1. Go to the Workflows page
  2. Click "New" button
  3. Select "From Template"
  4. Choose your "Simple Checklist" template
  5. Name it "My First Workflow"
  6. Click "Create Workflow"
3

Add Content to Your Workflow

Time: 1 minute

  1. You're now in Creation mode
  2. Click "+ Add Item" button
  3. Enter a name like "Review Policy Document"
  4. Add a description if needed
  5. Add 2-3 more items
  6. Click "Save Structure"
Remember: Always save your work! Changes aren't automatically saved.
4

Switch to Execution Mode

Time: 30 seconds

  1. Toggle the mode switch to "Execution"
  2. You'll see "Done" checkboxes appear
  3. Check off items as you complete them
  4. Click "Save Execution" to save your progress
5

Export to Board (Optional)

Time: 30 seconds

  1. In Execution mode, click "Export to Board"
  2. Choose export options (full or partial)
  3. Name your board
  4. Navigate to Boards to see your Kanban board
Congratulations! You've created your first template, workflow, and board. You're ready to explore more advanced features!

Common Questions

Do I need a template?

No! You can create an "Empty Workflow" to get started quickly. Templates are recommended for consistency when creating multiple workflows.

What's the difference between Creation and Execution?

Creation mode is for building structure (adding items, editing names). Execution mode is for tracking progress (checking items off, entering grades).

Can I change a workflow after creating it?

Yes! Switch back to Creation mode to add, edit, or delete items. Just remember to save your changes.
